Forum & Community

Ask questions, report bugs, suggest features, or share your experience with Jailer. There are two places to do that:


GitHub

The primary place for all Jailer discussions and issue tracking.

  • Discussions — Q&A, ideas, general conversation
  • Issues — bug reports and feature requests

A GitHub account is required to post.
SourceForge Forum Legacy

sourceforge.net/p/jailer/discussion/

The original Jailer forum. It is still active and supported, and offers anonymous posting.

New development activity and bug tracking has moved to GitHub. For bug reports and feature requests, GitHub Issues is preferred.

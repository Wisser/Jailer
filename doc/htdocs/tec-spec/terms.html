Jailer Terms


  • Entity E - a row inside a database table.
  • Reference {E1, E2}  - unordered pair of two Entities E1 and E2.
  • Association - set of references.
  • Join condition c of Association A - specifies an association between two tables:
                       A:={{E1, E2}|E1∈T1, E2∈T2, c({E1, E2})}
  • Restriction on an Association A - subset of A.
  • Restriction condition c - specifies a restriction R on A: R:={(E1, E2)|{E1, E2}∈A, c(E1, E2)}
  • Subject - set of Entities to be exported.
  • transitive closure of S - smallest set containing S and all entities referenced by an entity inside the closure. The transitive closure of the subject according to the resticted associations is exactly the set that Jailer will export.

HTML Data Model Rendering

Since 0.9.3 it is possible to generate a HTML render of the data model. A domain model is taken into account if it exists. Use the following command to produce a render of your data model:

$ sh bin/jailer.sh render-datamodel <jdbc-driver-class> <db-URL> <db-user> <db-password> {<restriction-model>}*

A data base connection is necessary to retrieve the schema of each table. Klick here to see an example.
